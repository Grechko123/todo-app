# TODO Приложение

Учебный проект: TODO-приложение с frontend (HTML + CSS + JavaScript) и backend (Node.js + Express + SQLite).

## Структура проекта

```
├── index.html          # Главная HTML-страница
├── style.css           # Стили приложения
├── script.js           # Frontend логика (чистый JavaScript)
├── backend/
│   ├── package.json    # Зависимости backend
│   ├── server.js       # Express сервер
│   ├── db.js           # SQLite база данных
│   └── routes.js       # API маршруты
└── README.md           # Документация
```

## Запуск

### Только Frontend (в браузере)

Просто откройте `index.html` в браузере. Приложение будет работать в локальном режиме — задачи хранятся в памяти JavaScript и пропадают при обновлении страницы.

### С Backend

1. Установите зависимости:
```bash
cd backend
npm install
```

2. Запустите сервер:
```bash
npm start
```

3. Откройте в браузере: `http://localhost:3000`

Приложение автоматически определит наличие backend и переключится на серверный режим.

## API Endpoints

| Метод  | URL               | Описание                        |
|--------|--------------------|---------------------------------|
| GET    | /tasks             | Получение задач (filter, page, limit) |
| POST   | /tasks             | Создание задачи                 |
| PATCH  | /tasks/:id         | Обновление задачи               |
| DELETE | /tasks/:id         | Удаление задачи                 |
| DELETE | /tasks/completed   | Удаление всех выполненных задач |

## Функциональность

- ✅ Добавление задач (кнопка + Enter)
- ✅ Редактирование задач (клик по тексту)
- ✅ Удаление задач (одиночное и массовое)
- ✅ Отметка выполнения (чекбокс)
- ✅ Выбрать все / снять все
- ✅ Фильтрация: Все / Активные / Завершённые
- ✅ Счётчики задач во вкладках
- ✅ Пагинация (5 задач на странице)
- ✅ Отмена ввода по Esc
- ✅ Отмена редактирования по Esc
- ✅ Ограничение длины задачи (250 символов)

## Защита от атак

- **SQL Injection**: Используются параметризованные запросы (prepared statements) в `db.js`
- **Stored XSS**: Текст задач экранируется функцией `sanitizeHtml()` в `routes.js` перед сохранением в БД
- **DoS через большие запросы**: Ограничение размера JSON-тела запроса (`10kb`) в `server.js`

## Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js, Express.js
- **База данных**: SQLite (better-sqlite3)
